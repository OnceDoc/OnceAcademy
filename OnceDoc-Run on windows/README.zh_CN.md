### 在 Windows 系统下运行 OnceDoc    

#### 第一步、安装 Node.js


#### 第二步、运行 OnceDoc

安装好 Node.js 后，我们可以运行 'oncedoc.release' 目录下的 start.cmd，启动 OnceDoc 的本地服务器：  
  
![运行 oncedoc.cmd][1]

#### 第三步、访问 OnceDoc

这时我们就可以在浏览器中访问 localhost:8064，使用 OnceDoc 的各项功能了：
